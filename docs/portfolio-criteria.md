# Portfolio Criteria

The goal is to build a job-relevant Full-Stack AI Engineer portfolio.

## Portfolio Project 1 — Chat with Documents

### Required Features

- document upload
- text extraction
- chunking
- embeddings
- vector search
- chat UI
- source references
- feedback buttons
- deployed demo
- clear README

### Quality Criteria

- explainable architecture
- tests for core logic
- clear error handling
- no hardcoded secrets
- cost and latency considerations
- documented limitations

## Portfolio Project 2 — AI Research Assistant

### Required Features

- controlled tool calling
- structured output
- state tracking
- logging
- guardrails
- clear final report

## Portfolio Project 3 — Prediction API

### Required Features

- dataset exploration
- model training
- evaluation metrics
- API endpoint
- tests
- deployment notes

## README Checklist

Each portfolio project should include:

- problem statement
- feature overview
- architecture diagram or text description
- setup instructions
- test instructions
- screenshots or demo link
- known limitations
- future improvements
